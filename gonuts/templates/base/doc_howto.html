Install nut tool

Make sure your GOPATH is correct (check output of go env). Then type 
go get -u github.com/AlekSi/nut/...
It will install nut binary into $GOPATH/bin. Then it's a nice idea to create empty config file ~/.nut.json – it will be updated by nut tool. 
touch ~/.nut.json
Check installation with 
nut help

Install a nut

nut get gonuts.io/vendor/nut
to install latest version, or 
nut get gonuts.io/vendor/nut/version
to install specific version. In both cases gonuts.io/ is optional. Package will be installed into $GOPATH/src/gonuts.io/vendor/nut, import path is gonuts.io/vendor/nut.

Publish a nut

To publish a package as nut on gonuts.io you should:
  • register on this site, choose a vendor name and copy access token to ~/.nut.json;
  • run nut generate inside package directory to generate spec.json, edit it;
  • run nut pack;
  • run nut publish [filename].
